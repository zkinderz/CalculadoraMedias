RESUMO

Este relatório apresenta o desenvolvimento de uma aplicação desktop denominada Calculadora de Médias Acadêmicas, criada com a linguagem C# e a tecnologia Windows Forms. O sistema tem como objetivo calcular a média semestral e a média final de um aluno, considerando as notas de NP1, NP2, PIM e Exame, além de informar automaticamente o status acadêmico do estudante.

O projeto foi estruturado em três partes principais: uma biblioteca de classes responsável pelas regras de negócio, uma aplicação desktop para interação com o usuário e um projeto de testes unitários utilizando xUnit. Também foram geradas evidências técnicas, como a DLL da biblioteca, o pacote NuGet, documentação em arquivos Markdown, diagrama UML, User Stories e versionamento do código no GitHub.

A solução desenvolvida permite validar entradas de dados, impedir notas inválidas, calcular médias com arredondamento adequado e exibir os resultados ao usuário de forma simples e objetiva. Dessa forma, o trabalho atende aos requisitos propostos, demonstrando a aplicação prática de conceitos de programação orientada a objetos, separação de responsabilidades, testes automatizados, documentação técnica e versionamento de software.

Palavras-chave: C#. Windows Forms. Testes Unitários. xUnit. NuGet. GitHub. Média Acadêmica.

1 INTRODUÇÃO

O uso de sistemas computacionais para automatizar cálculos acadêmicos contribui para a redução de erros manuais e facilita a análise do desempenho dos alunos. Em ambientes educacionais, o cálculo de médias é uma tarefa recorrente, sendo necessário considerar diferentes avaliações, pesos e critérios de aprovação.

Neste contexto, foi desenvolvido o projeto Calculadora de Médias Acadêmicas, uma aplicação desktop construída em C# com Windows Forms. O sistema permite informar as notas de NP1, NP2 e PIM para calcular a média semestral. Caso o aluno fique em exame, o sistema habilita o campo de nota do exame e calcula a média final.

Além da interface gráfica, o projeto foi estruturado com uma biblioteca de classes separada, responsável pelas regras de negócio. Essa separação permite maior organização, manutenção e reaproveitamento do código. Também foram implementados testes unitários com xUnit para validar os principais métodos do sistema.

O projeto contempla ainda documentação técnica, User Stories, diagrama UML, geração da DLL, criação de pacote NuGet e envio do código-fonte para o GitHub.

6.2 Regras de negócio

A média semestral é calculada considerando os pesos de 40% para NP1, 40% para NP2 e 20% para PIM. 

Média Semestral = NP1 × 0,4 + NP2 × 0,4 + PIM × 0,2

As regras de status semestral são:

  • Média Semestral maior ou igual a 7,0: Aprovado.
  • Média Semestral menor que 4,0: Reprovado.
  • Média Semestral entre 4,0 e 6,9: Em Exame.

Caso o aluno fique em exame, a média final é calculada pela média entre a média semestral e a nota do exame. 

Média Final = (Média Semestral + Exame) / 2

As regras de status final são:

  • Média Final maior ou igual a 5,0: Aprovado.
  • Média Final menor que 5,0: Reprovado.
